4.1 CSS — globals.css Dark Mode Overrides

Reemplazar las líneas 97-158 de src/app/globals.css con:

/* Reemplazar bloque .dark existente (líneas 97-158) */ .dark { /* ── Escala Neutra (Slate Frío - Dark Overrides) ── */ --n-1: #0b0f15; /* page bg */ --n-2: #11161f; /* surface-75 */ --n-3: #161c27; /* surface-100 */ --n-4: #1c2330; /* surface-200 */ --n-5: #232b3a; /* border-muted */ --n-6: #2c3648; /* border-default */ --n-7: #3a4558; /* border-strong */ --n-8: #596376; /* muted text */ --n-9: #7a8496; /* secondary text */ --n-10: #8a97ab; /* strong text (reducido) */ --n-11: #9faab9; /* heading (reducido de #cbd2de) */ --n-12: #b8c4d0; /* ink (reducido de #eef1f5) */ /* ── Brand Dark ── */ --b-8: #1fd4a8; --b-9: #1ac496; /* ── Semantic Colors Dark ── */ --s-info: #60a5fa; } .dark { /* ── Escala Neutra (Gris Cálido - Dark) ── */ /* Inspirado en Supabase Dark: #0f0f0f base */ --n-1: #090d13; /* page bg - negro cálido sin tinte */ --n-2: #111318; /* surface-75 */ --n-3: #181b23; /* surface-100 */ --n-4: #1e222b; /* surface-200 */ --n-5: #262a36; /* border-muted */ --n-6: #2e3340; /* border-default */ --n-7: #3d4250; /* border-strong */ --n-8: #5a6178; /* muted text */ --n-9: #7a8496; /* secondary text */ --n-10: #8a97ab; /* strong text */ --n-11: #9faab9; /* heading */ --n-12: #c0cad6; /* ink */ /* ── Brand Dark (desaturado 5%) ── */ --b-8: #1ac496; --b-9: #16b085; /* ── Semantic Colors Dark ── */ --s-success: #1ac496; --s-success-bg: rgba(26, 196, 150, 0.12); --s-success-br: rgba(26, 196, 150, 0.3); --s-warning: #f59e0b; --s-warning-bg: rgba(245, 158, 11, 0.12); --s-warning-br: rgba(245, 158, 11, 0.3); --s-danger: #ef4444; --s-danger-bg: rgba(239, 68, 68, 0.12); --s-danger-br: rgba(239, 68, 68, 0.3); --s-info: #4a9eff; /* teal-ish blue en lugar de azul puro */ --s-info-bg: rgba(74, 158, 255, 0.12); --s-info-br: rgba(74, 158, 255, 0.3); --s-neutral: #9ca3af; --s-neutral-bg: rgba(156, 163, 175, 0.12); --s-neutral-br: rgba(156, 163, 175, 0.3); }

4.2 FOCUS VISIBLE — Restaurar Accesibilidad

CRÍTICO: Reemplazar líneas 164-172 de globals.css:

/* ── Neutralizar focus ring del navegador ── */ :focus-visible { outline: none; box-shadow: none; } .dark :focus-visible { outline: none; box-shadow: none; } /* ── Focus ring accesible (WCAG 2.4.7) ── */ :focus-visible { outline: 2px solid var(--b-8); outline-offset: 2px; } /* Componentes con ring personalizado mantienen su estilo */ .dark :focus-visible { outline: 2px solid var(--b-8); outline-offset: 2px; box-shadow: 0 0 0 4px rgba(26, 196, 150, 0.2); /* Glow sutil */ }

4.3 prefers-color-scheme — Detección Automática

Agregar al final del bloque :root en globals.css:

/* ── Detectar preferencia del sistema operativo ── */ @media (prefers-color-scheme: dark) { :root:not(.light-only) { color-scheme: dark; } }

Nota: El theme provider actual (next-themes) maneja esto automáticamente cuando usa attribute: 'class'. Verificar que no haya conflicto si se usa class + media query.
